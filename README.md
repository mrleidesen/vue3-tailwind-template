# vue3-windi-template

已经集成`Vite2`+`Vue3`+`VueRouter4`+`Windi CSS`，其余如需要根据 vite 文档来进行配置

## 下载

- 直接通过`Download ZIP`
- 通过 Git clone

```
git clone https://github.com/mrleidesen/vue3-windi-template.git
```

## 运行

```
npm install
npm run dev

# or yarn

yarn
yarn dev
```
